# Release Note 0.6.38 #

## Neu
+ interactionType: WRITE: 
  + Parameter wurden aktualisiert. Es ist jetzt möglich, die Tastatur mit Groß- oder Kleinbuchstaben zu verwenden.
+ interactionType: DROP: 
  + Der Parameter buttonType wurde mit SMALL_SQUARE (170px) und EXTRA_SMALL_SQUARE (150px) hinzugefügt.
+ Stiländerungen für interactionType: DROP und EQUATION:
  + Wenn sich oben ein Bild befindet, wird der 90px-Abstand entfernt.
